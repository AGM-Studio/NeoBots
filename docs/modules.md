### Withdraw Module
<img src="https://raw.githubusercontent.com/AGM-Studio/NeoBots/refs/heads/main/src/main/resources/assets/create_neobots/textures/item/andesite_withdraw_module.png" width="64" height="64" alt="">
<img src="https://raw.githubusercontent.com/AGM-Studio/NeoBots/refs/heads/main/src/main/resources/assets/create_neobots/textures/item/brass_withdraw_module.png" width="64" height="64" alt="">

> Orders the bot to withdraw items from the selected container.  
> * Right click a container to set the target.  
> * Right click in air to access GUI configuration for more customization.
> > **ProTip:** Accessing a depot is faster than any other container.
>
> > **Brass Tier:** Brass module has built-in pathfinding to the container.

### Charging Module
<img src="https://raw.githubusercontent.com/AGM-Studio/NeoBots/refs/heads/main/src/main/resources/assets/create_neobots/textures/item/andesite_charging_module.png" width="64" height="64" alt="">
<img src="https://raw.githubusercontent.com/AGM-Studio/NeoBots/refs/heads/main/src/main/resources/assets/create_neobots/textures/item/brass_charging_module.png" width="64" height="64" alt="">

> Allows the bot to connect to a charging pad and charge while standing on top of it.
> > **Andesite Tier:** has no configuration and the bot must be standing on top of a pad to work.  
> > **Brass Tier:** Brass module has built-in pathfinding to the selected charging pad. Has GUI configurations.
> > * Right click a charging pad to set the target.
> > * Right click in air to access GUI configuration for more customization.
